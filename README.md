# Kütüphane CRUD Uygulaması

React tabanlı Vite framework ile geliştirilmiş, Bootstrap 5 destekli ve LocalStorage kullanan kütüphane yönetim uygulaması.

## 🚀 Özellikler

✅ Kitap ekleme  
✅ Kitap listeleme  
✅ Kitap güncelleme  
✅ Kitap silme  
✅ LocalStorage ile verileri saklama  
✅ Bootstrap 5 ile responsive tasarım  
✅ Netlify'a deploy edilebilir  

## 📦 Kurulum

```bash
# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev
